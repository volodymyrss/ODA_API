Installation
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1) Anaconda
     .. code-block:: bash

        - conda  install --file requirements.txt
        - python setup.py install

2) PIP
    .. code-block:: bash

        - pip install -r requirements.txt
        - python setup.py install`
